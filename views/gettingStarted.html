1

Download

After you click download, go through the application install process. It will give you the ability to run the command forgit CLI anywhere in your shell.

Mac

Linux (beta)

2

Settings

When you logged in you were on the Terminal page. In the second section you will have a default setting group. This is where you can set your preferred settings. You can also create multiple setting groups with different settings. You can only choose one setting group at a time.

So go set your settings you want if you haven't already. You can edit the default setting group.

3

Initialize

$ forgit init

UUID: {{.User.ForgitID}}

If this is the first setup on this computer forgit init will be required. The initialization command will grab all your settings you have set on the Terminal page. It will configure your Forgit directory and .forgitConf.json file for you on your local machine.

When you run this command you will be asked for the path you want the Forgit directory to live on your computer. Next you will be asked for the UUID on the Terminal page. At that point you will be ready to move to the start command.

4

Start

$ forgit start

Since this app can run offline, you can give this command a few different parameters.

If a -c [Minutes] or -p [Minutes] is passed it will automate all repos in the Forgit directory just for the one session. You can select one of your setting groups if you would like instead.

Commit Setting

[ -c MINUTE ]

Push Setting

[ -p MINUTE ]

Setting Group

[ g EXACT_NAME ]

Examples :

$ forgit start -c 10 -p 60
$ forgit start g Example

5

Stop

$ forgit Stop

This command will just tell you to use control-c, or close the shell window. If you do not stop the app it will continue to run until you shutdown the computer.
